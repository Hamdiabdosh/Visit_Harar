## Quick reference

### Core mission

**One place where any tourist can find everything they need** — inspire, discover, plan, book, and navigate Harar.

### Strategic decision

**Customize and evolve** the existing TanStack Start + PostgreSQL + CMS platform. Do not rebuild from scratch.
